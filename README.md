# 截图工具

一款轻量级 Windows 截图工具，支持屏幕截取、区域标注、文字识别（OCR）等功能。基于 C# WinForms 开发，无需安装 .NET 运行时，解压即用。

## 功能概览

### 截图

- 全屏截取，快捷键一键触发
- 鼠标拖拽选取任意矩形区域
- 实时显示选区尺寸与光标处像素颜色（RGB）
- 双击选区直接复制到剪贴板

### 标注

截取屏幕后，可在选区内添加以下标注：

| 标注类型 | 说明 |
|---------|------|
| 矩形 | 绘制矩形边框 |
| 椭圆 | 绘制椭圆边框 |
| 箭头 | 绘制带拖尾渐变的填充箭头，头部随长度等比放大（500px 封顶） |
| 画笔 | 自由手绘线条 |
| 文字 | 添加文字标注，支持字号与颜色调整 |
| 高亮 | 半透明矩形高亮覆盖 |
| 马赛克 | 像素化遮盖敏感内容 |

所有标注均支持：

- 拖动调整位置
- 矩形/椭圆/高亮/马赛克/文字支持 8 个手柄调整大小
- 文字标注双击进入编辑
- Delete 键删除选中标注
- 选中标注时滑动条联动属性（粗细/字号）

### 文字识别（OCR）

- 基于 Tesseract 引擎，支持中英文识别
- 提供 6 种 PSM 识别模式可选（PSM 3/4/6/7/8/11）
- 图像预处理：2 倍放大 + 灰度化，提升识别准确率
- 识别结果弹出独立窗口，可拖选复制
- 零安装模式：将 Tesseract 放入程序同目录 `ocr/` 即可使用

### 其他

- 颜色面板：8 色快速切换（红/蓝/绿/黄/橙/紫/黑/白）
- 粗细/字号滑动条：非文字工具调节线条粗细（1~8px），文字工具调节字号（10~64pt）
- 另存为：保存标注后的图片到文件
- 复制并关闭：将标注结果复制到剪贴板
- 撤销：撤销最近一次标注操作
- 右键菜单：提供标注操作的快捷入口

## 使用方式

### 启动

双击 `截图工具.exe`，程序将在系统托盘区驻留图标。

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Q（默认，可自定义） | 触发截图 |
| F1（固定备用） | 触发截图 |
| Esc | 取消截图 / 取消选中标注 |

快捷键可通过右键托盘图标 → 设置快捷键进行自定义，支持 Ctrl/Alt/Shift 与字母键的组合。

### 截图流程

1. 按下快捷键，屏幕进入截图模式（全屏半透明遮罩）
2. 鼠标拖拽选取目标区域
3. 使用工具栏添加标注
4. 点击「复制」或「另存为」完成操作

## 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 7 及以上（32/64 位均支持） |
| 运行时 | 无需额外安装，.NET Framework 4.0 已内置于系统 |

## 编译

如需从源码编译，需确保系统已安装 .NET Framework 4.0 SDK：

```batch
C:\Windows\Microsoft.NET\Framework\v4.0.30319\csc.exe ^
    /target:winexe /win32icon:截图.ico /optimize+ ^
    /r:System.dll /r:System.Drawing.dll /r:System.Windows.Forms.dll ^
    /out:截图工具.exe ScreenshotTool.cs
```

或直接执行项目目录下的 `build.bat`。

## OCR 配置

如需文字识别功能，按以下步骤配置：

1. 下载 Tesseract OCR（Windows 7 请使用 4.x 版本，5.x 不支持 Win7）
2. 将 `tesseract.exe` 放入程序目录下的 `ocr/` 文件夹
3. 将中文语言数据文件 `chi_sim.traineddata` 放入 `ocr/tessdata/` 文件夹

完成后目录结构应为：

```
截图工具.exe
ocr/
  tesseract.exe
  tessdata/
    chi_sim.traineddata
```

若未配置 OCR，截图功能不受影响，仅文字识别按钮点击时提示未安装。

## 技术说明

- 开发语言：C# 5.0（兼容 csc.exe 命令行编译，不依赖 Roslyn）
- UI 框架：Windows Forms
- 目标框架：.NET Framework 4.0
- 单文件架构：全部代码集中在 `ScreenshotTool.cs`
- DPI 感知：声明 DPI 感知以获取真实物理像素
- 工具栏按钮图标：GDI+ 手绘生成，无需外部图片资源
